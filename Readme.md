1. <!-- var, let, const -->
Ans:-
var: function scoped, can be changed.

let: block scoped, can be changed.

const: block scoped, cannot be changed after assignment.

2. <!-- Spread Operator (...) -->
Ans:-
Expands arrays or objects into individual elements.
3. <!-- map(), filter(), forEach() -->
Ans:-
map(): returns a new array after transforming elements.
filter(): returns a new array with elements that meet a condition.
forEach(): loops through elements, returns nothing.

4. <!-- Arrow Function -->
Ans:-
Shorter syntax for writing functions.

5. <!-- Template Literals -->
Ans:-
Backticks ` used for strings with ${} to insert variables.
its very powerful.
